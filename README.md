# Reelax Frontend Assignment

A responsive React JS implementation of the Reelax checkout and billing dashboard based on the provided Figma design.

## 🚀 Tech Stack

- React JS
- Vite
- Tailwind CSS
- React Icons

## ✨ Features

- Responsive layout
- Reusable components
- Billing form section
- Order summary card
- Coupon section
- Modern UI styling

## 📦 Installation

```bash
npm install
npm run dev
```

## 🌐 Deployment

Deployed using Vercel.

## 📁 Folder Structure

```txt
src/
 ├─ assets
 ├─ components
 │   ├─ common
 │   ├─ layout
 │   └─ sections
 ├─ pages
 └─ App.jsx
```
